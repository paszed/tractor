# Tractor

Tractor is a simple CLI tool for scraping websites and exporting structured data (JSON/CSV).

Status: early development — nothing implemented yet.

This repository is just initialized and will be built step by step with a focus on:
- minimalism
- speed
- real-world usability

Planned:
- input: URL
- fetch HTML
- extract via CSS selectors
- output: JSON or CSV

No setup, no install, no usage yet.

Work in progress.
